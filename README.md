### Description

This is a program that has various ways of unloading a form... some ways are:

expand 1, 2, 3

diagonal UpperRight,UpperLeft,LowerRight,LowerLeft

Bounce Up,Down,Left,Right

Slide Up,Down,Left,Right

and a few more..
